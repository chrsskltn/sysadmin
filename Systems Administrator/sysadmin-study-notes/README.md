
# Sysadmin Study Notes

Self-directed learning in preparation for starting as Information Services Administrator at BCD Group on 8 April 2026.
## Goals
- Get productive with the Microsoft Power Platform (Power Automate, Power Apps, Power BI, Power Pages, Copilot Studio)
- Build solid SQL and relational database fundamentals
- Learn Git and version control workflows
- Get comfortable with SharePoint architecture and administration
- Develop basic scripting skills in PowerShell and Python
- Understand REST APIs and how to integrate them
- Learn M365 administration basics (Teams, OneDrive, Exchange)
- Understand security, compliance, and the NZ Privacy Act 2020
## Structure
| Folder            | Topic                                                |
| ----------------- | ---------------------------------------------------- |
| `git/`            | Version control fundamentals                         |
| `sql/`            | SQL Server and relational databases                  |
| `power-platform/` | Power Automate, Power Apps, Power BI, Copilot Studio |
| `sharepoint/`     | SharePoint sites, lists, libraries, permissions      |
| `scripting/`      | PowerShell and Python                                |
| `apis/`           | REST APIs and integrations                           |
| `m365/`           | Microsoft 365 admin                                  |
| `security/`       | Security, compliance, NZ Privacy Act                 |
